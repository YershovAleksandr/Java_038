Отправка почты осуществляется с адреса softtest42@74.ru
в config.properties указан этот же адрес в качестве получателя (это yandex почта, пароль: softaria).

Скрипт init.sh с помощью curl-а создаёт несколько файлов для дальнейшего использования
Программа считывает с консоли последовательно:
1) Целое число - количество записей на вчера.
2) Соответствующее количество пар строк - url и имя файла.
3) Целое число - количество записей на сегодня.
4) Соответствующее количество пар строк - url и имя файла.

Пример:
2
url1 file1
url2 file2
1
url3 file3

Для удобства запуска параметры записаны в файле input.txt
Для запуска программы:
mvn compile exec:java < input.txt

Для запуска тестов:
mvn compile test
